Hello,
Rawposts is a free jekyll portfolio-style theme, which is designed to be as a starting point for any Jekyll website. By using this theme, the user can deploy the personal blog or website on Github pages easily.

This theme supports features:

  • Fully responsive
  • Optimized for tablets & mobiles
  • Pagination
  • Archive posts
  • Category posts
  • Gittalk
  • Disqus
  • Google analytics
  • busuanzi analytics
  • rss
  • jemoji for emoji
  • dark mode selectable
The theme is designed to be easily customizable (fonts, colors, layout elements, code coloration, etc...) and all above features are defined at _config.yml file. The users are able to turn on/off these features only by setting the key words at _config.yml easily.

For the installation & usage guidance, please refer to Readme.MD
